The YouVersion Platform provides powerful search functionality that allows you to find specific Bible content across multiple translations and languages.

## Overview

Search functionality enables you to:

- Find specific verses or passages
- Search for keywords across multiple Bible translations
- Get context around search results
- Filter results by Bible version, book, or language

## Search Endpoints

### Basic Search

Search for content across all available Bible versions:

```bash
curl -H "X-App-Id: YOUR_APP_ID" \
     "https://api.youversion.com/v1/bibles/111/search?q=love"
```

### Advanced Search Parameters

You can refine your search with additional parameters:

```bash
curl -H "X-App-Id: YOUR_APP_ID" \
     "https://api.youversion.com/v1/bibles/111/search?q=love&limit=10&offset=0"
```

### Search Response Format

```json
{
  "data": [
    {
      "reference": "John 3:16",
      "text": "For God so loved the world...",
      "bible": {
        "id": 111,
        "name": "New International Version",
        "abbreviation": "NIV"
      },
      "score": 0.95
    }
  ],
  "next_page_token": "string"
}
```

## Search Parameters

| Parameter | Type | Description | Default |
|-----------|------|-------------|---------|
| `q` | string | Search query | Required |
| `limit` | integer | Maximum results per page | 20 |
| `offset` | integer | Number of results to skip | 0 |
| `highlight` | boolean | Include highlighting in results | false |

## Examples

### Search for "faith" in NIV

```javascript
const response = await fetch(
  'https://api.youversion.com/v1/bibles/111/search?q=faith',
  {
    headers: {
      'X-App-Id': 'YOUR_APP_ID'
    }
  }
);

const results = await response.json();
console.log('Search results:', results.data);
```

### Search with highlighting

```python
import requests

def search_with_highlighting(query, version_id=111):
    headers = {'X-App-Id': 'YOUR_APP_ID'}
    params = {
        'q': query,
        'highlight': True,
        'limit': 10
    }
    
    response = requests.get(
        f'https://api.youversion.com/v1/bibles/{version_id}/search',
        headers=headers,
        params=params
    )
    
    return response.json()

# Usage
results = search_with_highlighting('grace')
for result in results['data']:
    print(f"{result['reference']}: {result['text']}")
```

## Best Practices

1. **Use specific queries**: Instead of searching for "God", try "God's love" or "God created"
2. **Limit results**: Use the `limit` parameter to control response size
3. **Handle pagination**: Use `next_page_token` for large result sets
4. **Cache results**: Implement caching for frequently searched terms
5. **Error handling**: Always handle potential search errors gracefully

## Error Handling

Search requests can return various error responses:

```javascript
try {
  const response = await fetch(searchUrl, {
    headers: { 'X-App-Id': 'YOUR_APP_ID' }
  });
  
  if (!response.ok) {
    if (response.status === 400) {
      throw new Error('Invalid search query');
    } else if (response.status === 401) {
      throw new Error('Authentication required');
    } else if (response.status === 429) {
      throw new Error('Rate limit exceeded');
    }
  }
  
  const results = await response.json();
  return results;
} catch (error) {
  console.error('Search failed:', error);
  return { data: [], error: error.message };
}
```

## Rate Limiting

Search requests are subject to rate limiting. Check response headers for rate limit information:

- `X-RateLimit-Limit`: Maximum requests per time window
- `X-RateLimit-Remaining`: Remaining requests in current window
- `X-RateLimit-Reset`: Time when the rate limit resets

## Integration Examples

### React Hook for Search

```jsx
import { useState, useEffect } from 'react';

function useBibleSearch() {
  const [results, setResults] = useState([]);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState(null);

  const search = async (query, versionId = 111) => {
    setLoading(true);
    setError(null);
    
    try {
      const response = await fetch(
        `https://api.youversion.com/v1/bibles/${versionId}/search?q=${encodeURIComponent(query)}`,
        {
          headers: { 'X-App-Id': 'YOUR_APP_ID' }
        }
      );
      
      if (!response.ok) {
        throw new Error(`Search failed: ${response.status}`);
      }
      
      const data = await response.json();
      setResults(data.data);
    } catch (err) {
      setError(err.message);
    } finally {
      setLoading(false);
    }
  };

  return { results, loading, error, search };
}
```

### Search Component

```jsx
function SearchComponent() {
  const [query, setQuery] = useState('');
  const { results, loading, error, search } = useBibleSearch();

  const handleSubmit = (e) => {
    e.preventDefault();
    if (query.trim()) {
      search(query);
    }
  };

  return (
    <div>
      <form onSubmit={handleSubmit}>
        <input
          type="text"
          value={query}
          onChange={(e) => setQuery(e.target.value)}
          placeholder="Search Bible..."
          disabled={loading}
        />
        <button type="submit" disabled={loading}>
          {loading ? 'Searching...' : 'Search'}
        </button>
      </form>

      {error && <p className="error">{error}</p>}

      {results.length > 0 && (
        <div className="results">
          {results.map((result, index) => (
            <div key={index} className="result">
              <h3>{result.reference}</h3>
              <p>{result.text}</p>
              <small>{result.bible.name}</small>
            </div>
          ))}
        </div>
      )}
    </div>
  );
}
```
